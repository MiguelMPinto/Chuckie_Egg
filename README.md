# Chuckie Egg

Kotlin implementation of **Chuckie Egg**, developed as an academic programming project at ISEL.

## Overview

This repository contains a desktop game implementation inspired by Chuckie Egg. The project focuses on game logic, Kotlin/JVM programming and graphical rendering through a lightweight canvas library.

## Main features

- Chuckie Egg-style gameplay
- Kotlin/JVM implementation
- Desktop application entry point
- Canvas-based rendering
- Gradle project structure

## Tech stack

- Kotlin
- Gradle
- CanvasLib JVM
